# escuela-reverse-proxy

For Posthog custom domain instead of theirs

<https://posthog.com/docs/advanced/proxy/cloudflare>

```
npx wrangler deploy
```

## Deployment

To deploy this project:

1. Clone this git repo
1. Install [wrangler2](https://developers.cloudflare.com/workers/wrangler/get-started/).
1. Run `wrangler publish`
